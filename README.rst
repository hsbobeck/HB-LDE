HB-LDE
======

HB-LDE is a fork of `LDE (Louper Desktop Encoder) <https://www.louper.io/lde>`_ v2.0.7,
which is itself a fork of `OBS Studio <https://obsproject.com>`_.

The primary modification is the ability to stream to any domain by defining a custom Stream URL and stream key.
