HB-LDE
======

HB-LDE is a fork of `LDE (Louper Desktop Encoder) <https://louper.io>`_ v2.0.7,
which is itself a fork of `OBS Studio <https://obsproject.com>`_.

HB-LDE v1.0.0 is based on LDE v2.0.7. It retains the same core capabilities:
video capture, compositing, encoding, recording, and live streaming.
